# How to setup:

$ pip install -r requirements.txt

open the config.py

change the server_host on your ip

![image](https://user-images.githubusercontent.com/101527966/174895939-564db7cc-cb90-436c-a8ca-5df0c8e7b005.png)

$ cd server

$ python start.ru
